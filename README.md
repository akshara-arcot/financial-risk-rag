# Financial Risk RAG Assistant

A Retrieval Augmented Genertaion (RAG) application that answers financial risk questions from annual reports and 10-K filing forms using vector search and LLM reasoning.
Built a modular Financial Risk RAG system with FastAPI backend and Docker deployment to simulate production ML workflows.

## Live Demo

Streamlit App:
https://financial-risk-rag-wveuvhihfkcqlytgp2wf7q.streamlit.app

## Features

- Retrieval Augmented Generation (RAG)
- Financial document analysis
- Sentence Transformer embeddings
- LLM powered response generation
- Streamlit UI for interaction
- FastAPI backend for API access
- Docker container support
